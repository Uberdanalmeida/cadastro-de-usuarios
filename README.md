# 🚀 Cadastro de Usuários Full Stack

Bem-vindo ao meu projeto de Cadastro de Usuários! Este é um sistema completo onde o front-end conversa com um servidor real, que por sua vez guarda as informações em um banco de dados na nuvem. Adeus, dados que sumiam ao dar refresh! 😎

---

## 💡 O que este projeto faz?

* **🌐 Interface Moderna**: Criada com React e estilizada para ser intuitiva.
* **🔌 API Própria**: Backend construído em Node.js com Express.
* **🗄️ Banco de Dados Real**: Integração com o Supabase (PostgreSQL) para persistência total de dados.
* **🔐 Segurança**: Uso de variáveis de ambiente (`.env`) para proteger chaves de acesso.
* **🗑️ CRUD Completo**: Você pode listar, cadastrar e deletar usuários em tempo real.
